# Portfólio Pessoal

Landing page de portfólio pessoal desenvolvida com **HTML, CSS e JavaScript puro**, com foco em identidade visual, navegação simples e animações na seção principal.

## ✨ Funcionalidades atuais

- Header fixo com navegação por âncoras
- Seção Hero com apresentação pessoal
- Links para redes sociais
- Botões de ação (CTA)
- Efeito de digitação no subtítulo (`Fullstack Developer`)
- Cursor animado/piscante
- Estrutura inicial para seções de Sobre, Skills e Projetos

## 🧱 Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)
- Google Fonts (Poppins)
- Fontes locais (Clash Display)

## ▶️ Como executar localmente

Como é um projeto estático, você pode abrir o arquivo `index.html` direto no navegador.

Se quiser uma experiência melhor (recomendado), use o **Live Server** no VS Code:

1. Abra a pasta do projeto no VS Code
2. Clique com o botão direito em `index.html`
3. Selecione **Open with Live Server**
